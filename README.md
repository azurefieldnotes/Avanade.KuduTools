# Avanade.KuduTools
A simple set of PowerShell REST wrappers for Azure Websites Kudu console

## Functionality
* Virtual Filesystem (VFS)
    * Filesystem Info
        * File download links
    * Upload Files
* Settings
* Deployments
* Execute Commands
